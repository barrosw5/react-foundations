# React Foundations - Next.js

This repository contains the code developed during the Vercel React Foundations course. The project demonstrates the consolidation of fundamental Frontend development concepts, illustrating the transition from plain JavaScript to React, and subsequently to the Next.js framework.

The code covers component creation, state management, and an initial introduction to the Next.js App Router.
